# vi/bash 교육

## function과 subroutine 차이

프로그래밍에서 **함수(Function)**와 **서브루틴(Subroutine)**은 재사용 가능한 코드 블록이라는 공통점이 있지만, 가장 큰 차이는 **반환값(Return Value)의 유무**입니다.

### 주요 차이점

| 구분 | 함수 (Function) | 서브루틴 (Subroutine) |
| --- | --- | --- |
| 반환값 | 계산 결과나 처리된 값을 **항상 반환** | 값을 반환하지 않고 **명령만 수행** |
| 용도 | 수식의 일부로 사용되거나 결과를 얻기 위해 사용 | 출력, 상태 변경 등 특정 작업 반복 |
| 특징 | 수학적 함수 개념과 유사 | |

### 상세 설명

- **함수 (Function)**: 입력을 받아 연산 후 결과를 반환합니다. 예: `sum(a, b)`는 두 수를 더한 **결과값**을 반환합니다. 현대 언어에서는 반환값이 없더라도(`void`) 모두 '함수'로 통칭하는 경우가 많습니다.
- **서브루틴 (Subroutine)**: 로직을 모아둔 코드 블록입니다. 값을 반환하기보다 "로그 출력", "데이터베이스 저장"처럼 **특정 동작 수행**이 목적입니다. Fortran, Visual Basic 등에서 명확히 구분합니다.

## bash 함수

- function과 subroutine을 구분하지 않음
- `func1() { echo "hello, $1"; }`
- `func1 "world"`
- `function func2 { return $(($1 + $2)); }`
- `func2 3 4`
- `echo $?`
- 함수도 외부 실행 프로그램처럼 간주됨
